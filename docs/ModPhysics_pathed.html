Introduction

Hello! Welcome to the patched version of Mod Physics. This mod is designed for mobile users, including Pojavlauncher, HMCL, and others.

Information

  • This Mod is compatible with fabric 1.20.1.
  • Does not support any shader if you are using Optifabric or Iris.
  • Both Pro and Free versions are included.

Attention

This mod doesn't support Optifine Shaders, whether it's a pro or original version. Attempting to force it may result in crashes unless you use a zink renderer.
